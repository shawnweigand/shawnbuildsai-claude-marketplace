# Research Query Templates

Use these search query patterns when gathering intelligence for each competitor. Replace `[COMPANY]` with the competitor's name or domain.

## Blog Posts & Recent Content

```
[COMPANY] blog [YEAR]
[COMPANY] announcement site:[domain]
[COMPANY] news after:[YEAR]-[LAST_MONTH]-01
"[COMPANY]" new article OR post OR announcement
```

*Replace `[YEAR]` with the current year and `[LAST_MONTH]` with the prior month (e.g., `2026-02`).*

## Product Updates & Changelogs

```
[COMPANY] product update OR release OR changelog [YEAR]
[COMPANY] new feature OR launch OR pricing [YEAR]
site:[domain]/changelog OR /releases OR /whats-new
[COMPANY] pricing page
```

## Events & Webinars

```
[COMPANY] webinar OR event OR conference [YEAR]
[COMPANY] product launch event
"[COMPANY]" upcoming event site:[domain]
```

## Social Media

```
[COMPANY] site:linkedin.com
[COMPANY] site:twitter.com OR site:x.com
"[COMPANY]" announcement OR launch LinkedIn [YEAR]
```

## Homepage / Positioning

- Visit the company's homepage directly
- Note the H1 headline, subheadline, and primary CTA button text
- Check the "About" page for updated mission or positioning language
- Check the navigation for new product or solution pages

## Tips

- Use `after:YYYY-MM-DD` in Google search to filter by recency
- Check Google's "News" tab for press coverage
- For pricing: visit `/pricing` or `/plans` directly on their domain
- For product updates: check `/blog`, `/changelog`, `/releases`, `/updates`
